# Portfolio Customization Guide

## Change profile photo
Replace this file with your real photo:

`public/images/myphoto.png`

Keep the same file name, or update the path in `components/About.tsx`.

## Change CV
Replace this file with your real CV:

`public/cv.pdf`

## Change name and About text
Edit:

`components/About.tsx`

Search for `Regaltos` and change it to your real name.

## Change contact details
Edit:

`components/Contact.tsx`

Search for `yourmail@gmail.com`, `+91 XXXXX XXXXX`, and social links.

## Run project

```bash
npm install
npm run dev
```

Then open: http://localhost:3000
